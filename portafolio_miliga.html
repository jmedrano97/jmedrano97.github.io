Detalles

Utilizando un template html de uso libre como punto de partida estoy desarrollando esta web. Para darle funcionalidad, he utilizado Python. Con Django, he desarrollado la gestión de usuarios, la autenticación, la gestión de equipos y la programación de partidos, etc. Para facilitar el proceso de descarga y configuración de mi proyecto, he utilizado la base de datos SQLite que viene integrada en el framework Django.

Detalles Tecnicos

Además de utilizar vistas basadas en funciones y realizar CRUD's para implementar las funcionalidades de mi proyecto de liga de fútbol, también he creado un algoritmo personalizado para el emparejamiento de los encuentros. Este algoritmo utiliza técnicas de optimización para garantizar una distribución equitativa de los equipos y generar una lista de partidos que maximice la variedad y la emoción de la liga.

Por otro lado, para garantizar la integridad y la coherencia de los datos en la base de datos SQLite, he utilizado el sistema de transacciones de Django. De esta forma, cualquier modificación en la base de datos se ejecuta como una única operación atómica, lo que asegura que los cambios se completen en su totalidad o se deshagan por completo en caso de algún error o problema en el proceso.
